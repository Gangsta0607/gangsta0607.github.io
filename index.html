Программа для упрощения жизни студента. Может хранить предметы, расписания (в том числе возможность добавления альтернативного), расписания звонков (в том числе на каждый день недели), долгов по предметам (с датой сдачи) и прочими удобными штуками.

Краткая справка по работе с приложением

Для начала надо перейти на вкладку Предметы (вторая) и добавить предметы. После добавления предметов нужно перейти на вкладку Расписание (третья) и создать расписание, после этого в той же вкладке нужно добавить расписание звонков (маленькая плавающая кнопка снизу справа). После этого на главном экране во время пар будет показано расписание дня, и текущая пара будет выделена. Чтобы добавить долги по какому-либо предмету, надо перейти на вкладку Долги (четвёртая) и добавить нужный долг. После пар на главном экране будут показаны расписание и долги на следующий день, если они есть.
